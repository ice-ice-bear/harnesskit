# Git Workflow

## Commits
- Conventional commits: `feat:`, `fix:`, `refactor:`, `test:`, `docs:`
- One logical change per commit
- Write commit messages in imperative mood

## Branches
- `main` — production-ready
- `feat/{name}` — feature branches
- `fix/{name}` — bug fix branches
